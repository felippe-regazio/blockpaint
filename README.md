# Blockpaint

This is a super simple Pixel Art Editor made in an old fashioned architecture.

## Usage

Visit , or clone this repository and drop the `index.html, script.js, style.css` somewhere. Now just open the `index.html`.

## Features

- Multisize grid
- Share using URL
- Save the art
- Load saved art
- Colorpicker
